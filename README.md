# River Land Game (Python)

A Python mini-project that combines CSV data processing with a matrix-based river length game algorithm.

## Highlights
- Reads and processes training CSV data
- Uses dictionary-based authentication/data handling
- Computes river lengths in a user-defined matrix
- Includes simple game-style interaction logic

## Tech Stack
- Python
- CSV module

## Project Files
- `read_csv.py` - Parses CSV and extracts key fields
- `authenticate_data.py` - Dictionary-based data processing
- `game_functionality.py` - Matrix traversal and river-length logic

## Run Locally
```bash
python read_csv.py
python authenticate_data.py
python game_functionality.py
```
